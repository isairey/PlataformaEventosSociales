# 🌌 Acerca del proyecto

**Event Rental Platform** es una plataforma web diseñada para gestionar eventos sociales, reservaciones y servicios de renta desde una experiencia moderna e intuitiva.

El sistema permite administrar eventos como:

- 🎉 Fiestas privadas
- 💍 Bodas
- 🏢 Eventos corporativos
- 🎂 Cumpleaños
- 🎓 Graduaciones
- 🎵 Eventos musicales
- 🍽️ Banquetes
- 🪩 Celebraciones sociales

# 👨‍💼 Módulos del sistema

## 🎊 Event Module

Este módulo administra todos los eventos registrados en la plataforma.

### Funcionalidades:

- ➕ Crear eventos
- 📅 Gestionar fechas
- 📍 Seleccionar ubicaciones
- 📋 Configurar detalles
- 📊 Supervisar reservaciones

---

## 🪑 Rental Module

Este módulo administra productos y servicios de renta.

### Funcionalidades:

- 🏕️ Gestión de carpas
- 🪑 Administración de mobiliario
- 💡 Decoración e iluminación
- 🎵 Equipos multimedia
- 📦 Paquetes para eventos

---

## 👤 Customer Module

Este módulo es utilizado por clientes que desean reservar servicios.

### Funcionalidades:

- 📅 Reservar eventos
- 💳 Gestionar pagos
- 📄 Consultar historial
- 📞 Contactar administradores
- 📊 Ver estado de reservaciones

- npm

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/PlataformaEventosSociales.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd PlataformaEventosSociales
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Ejecutar entorno de desarrollo

```bash
npm run dev
```

---

## 5️⃣ Abrir aplicación

```bash
http://localhost:5173
```
